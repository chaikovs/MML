subfuns

PURPOSE

SUBFUNS lists all function declaration lines in specified MFILE.

SYNOPSIS

function SubFuns(mFile)

DESCRIPTION 

SUBFUNS lists all function declaration lines in specified MFILE.
   SUBFUNS(MFILE) displays list to Command Window.

Example:
   SubFuns SubFuns

Note:
   This utility uses both subfunctions and a nested function, both of
   which are supported.

See also FUNCTION.
