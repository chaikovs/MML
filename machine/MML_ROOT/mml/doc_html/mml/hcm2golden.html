hcm2golden

PURPOSE

HCM2GOLDEN - Set horizontal corrector strengths to golden values

SYNOPSIS

function hcm2golden(nstep)

DESCRIPTION 

HCM2GOLDEN - Set horizontal corrector strengths to golden values

 hcm2golden(nstep)

 INPUTS
 1. nstep - number of step for setting correctors {Default: 5}

 See also vcm2zero, vcm2golden, hcm2zero
