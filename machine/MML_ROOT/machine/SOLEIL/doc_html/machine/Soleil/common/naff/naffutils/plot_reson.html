DESCRIPTION 

 function plot_reson(a,b,c,fen)
 plot resonance whose equation is: a*fx+ b*fz = c 
 in a windows whose dimensions are in fen=[xmin xmax zmin zmax]
 eg:
 plot_reson(3,1,65,[18 19 10 11])
