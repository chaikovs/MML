DESCRIPTION 

 function reson(ordre, period, fen)
 Recherche des droites de resonance a x + b y = c
 passant dans la fenetre [x1 x2 y1 y2]
 Ordre de la resonance |a| + |b|
 Critere de recherche c = period * entie
 eg:
 [k, tab] = reson(4,1,[18 19 10 11]
