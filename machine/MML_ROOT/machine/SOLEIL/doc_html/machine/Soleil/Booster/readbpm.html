readbpm

PURPOSE

READBPM - Gets turn by turn sample data for one BPM

SYNOPSIS

function [X Z Q Sum Va Vb Vc Vd] = readbpm(varargin)

DESCRIPTION 

 READBPM - Gets turn by turn sample data for one BPM 

  INPUTS
  1. num - bpm number 
  Optionnal
  Display/NoDisplay

  OUTPUTS
  1. X - Horizontal data
  2. Z - Vertical data
  3. Q - Quadrupole signal data
  4. Sum - Sum signal data
  5. Va electrode data
  6. Vb electrode data
  7. Vc electrode data
  8. Vd electrode data
